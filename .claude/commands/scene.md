# scene - 印象的なシーンを作成する

物語の中の一場面を、五感に訴える豊かな描写で作成します。

## 使い方

```
/scene [タイプ] [オプション]
```

## シーンタイプ

- `dialogue` - 会話中心のシーン
- `action` - アクション中心のシーン
- `emotion` - 感情描写中心のシーン
- `transition` - 場面転換シーン
- `climax` - クライマックスシーン

## オプション

- `--character <名前>` - 使用するキャラクター指定
- `--mood <雰囲気>` - シーンの雰囲気（緊張、穏やか、etc）
- `--length <文字数>` - シーンの長さ
- `--purpose <目的>` - このシーンの役割

## シーン作成の基本構造

### 1. 状況設定（セットアップ）
```
【外的要素】
- 場所：具体的に、読者が想像できるように
- 時間：時刻、季節、天候
- 登場人物：誰がいて、どんな位置関係か

【内的要素】
- 視点人物の心理状態
- このシーンに入る前の出来事
- 期待と不安
```

### 2. シーンの展開
```
導入（フック）：読者を引き込む最初の一文
　↓
展開（ビルドアップ）：緊張や期待を高める
　↓
転換（ターニングポイント）：何かが変わる瞬間
　↓
締め（余韻）：次への橋渡し
```

### 3. 描写のバランス
```
視覚：40%（見えるもの）
聴覚：20%（聞こえるもの）
心理：30%（感じること）
その他：10%（触覚、嗅覚、味覚）
```

## 実行例

### 会話シーン
```
User: /scene dialogue --character 佐藤明日香 --mood 緊張
Assistant: 【会話シーンを作成します】

設定：
- キャラクター：佐藤明日香（図書館司書）
- 雰囲気：緊張
- 場所：図書館のカウンター

---

明日香は返却された本の山を前に、手を止めた。見覚えのない革表紙の手帳が、その中に混じっている。

「あの...」

カウンターの向こうで、青年が落ち着きなく指を組んでいた。

「これ、お客様のものですか？」

明日香が手帳を掲げると、青年の顔が見る間に赤くなった。

「それは...その...」

[心理：何か大切なものらしい。でも、なぜ本に紛れて？]

続けますか？
```

### 感情シーン
```
User: /scene emotion --character 田中美咲 --mood 懐かしさ
Assistant: 【感情描写シーンを作成します】

設定：
- キャラクター：田中美咲（書店店主）
- 感情：懐かしさ
- きっかけ：母の愛読書を見つける

---

埃をかぶった『銀河鉄道の夜』を手に取った瞬間、美咲の指が止まった。

ページの間から、かすかにラベンダーの香りが漂う。母の香水だ。二十年経った今でも、この香りは美咲を別の時代へと運んでいく。

「また宮沢賢治？」と笑いながら、母はよくこの本を抱えていた。病室のベッドサイドにも、最後までこの本があった。

美咲は本を胸に抱いた。固い表紙越しに、母の温もりを探すように。

[内面：涙は出ない。でも、胸の奥が静かに熱い]
```

### アクションシーン
```
User: /scene action --mood 緊迫
Assistant: 【アクションシーンを作成します】

雰囲気：緊迫
シーンの目的：危機的状況での決断

---

階段を駆け上がる足音が、古い建物全体に響いた。

三段飛ばしで上がりながら、翔は肩越しに振り返る。追手の影はまだ見えない。だが、時間の問題だ。

最上階のドアが目前に迫る。鍵は掛かっているはずだ。体当たりするか、それとも—

「待て！」

下から声が響いた。立ち止まれば終わりだ。

翔は全体重を込めて、ドアに肩をぶつけた。

[判断：今しかない]
```

## ワークスペースでのシーン管理

### シーンファイルの保存
```
# 個別シーンとして保存
my-stories/[作品名]/scenes/scene-[番号]-[内容].md

# 例
my-stories/last-letter/scenes/scene-01-bookstore.md
my-stories/last-letter/scenes/scene-02-letter-discovery.md
my-stories/last-letter/scenes/scene-03-revelation.md
```

### シーンの命名規則
- `scene-[番号]-[内容].md` 形式を推奨
- 番号は2桁（01, 02...）で統一
- 内容は英語またはローマ字で簡潔に

### 下書きの管理
```
# 実験的なシーンは drafts/ に
drafts/experimental-dialogue.draft.md
drafts/alternative-ending.wip.md

# 採用が決まったら正式な場所へ移動
mv drafts/experimental-dialogue.draft.md my-stories/story-name/scenes/scene-04-dialogue.md
```

### シーンの品質管理
各シーンを作成したら：
1. `/quality scene [シーン名]` で評価
2. 問題点を修正
3. 本編に組み込み

## シーンタイプ別テクニック

### 会話シーン
- 台詞の間に仕草を挟む
- 沈黙も効果的に使う
- サブテキスト（言外の意味）を意識

### 感情シーン
- 直接的な感情表現を避ける
- 行動や情景で感情を表現
- 五感を通じた記憶の描写

### アクションシーン
- 短い文で緊迫感を演出
- 動詞を効果的に使用
- 時間の流れを明確に

### 場面転換
- 時間経過を自然に示す
- 前シーンとの対比を活用
- 新しい情報を少しずつ開示

## 効果的な描写のコツ

### Show, Don't Tell
```
❌ 彼は怒っていた。
✅ 拳を握りしめ、顎の筋肉がぴくりと動いた。
```

### 比喩の活用
```
❌ とても静かだった。
✅ 図書館は、時が止まったように静まり返っていた。
```

### 感覚的描写
```
❌ 古い本がたくさんあった。
✅ カビと埃の匂いが鼻をつき、革装丁の背表紙が薄暗い光を吸い込んでいた。
```

💡 キャラクターの内面と外面のバランスを取る
💡 長すぎるシーンは分割を検討
